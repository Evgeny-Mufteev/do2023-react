# Начало работы с приложением React

### `npm i`

Установите зависимости проекта

## Доступные скрипты

В каталоге проекта вы можете запустить:

### `npm run all`

При запуске приложения дополнительно используется json-сервер на другом порту, чтобы имитировать API. Что бы не запускать два терминала с локальными серверами в каждом, используется библиотека npm-run-all которая запускает серверы в одном терминале параллельно.

Страница перезагрузится, если вы внесете изменения.\
Вы также увидите любые ошибки lint в консоли.

### `npm run build`

Создает приложение для производства в папке `build`.\
Он правильно объединяет React в производственном режиме и оптимизирует сборку для достижения наилучшей производительности.

Сборка минимизирована, а имена файлов включают хэши.\
Ваше приложение готово к развертыванию!

См. раздел о [развертывании](https://facebook.github.io/create-react-app/docs/deployment) для получения дополнительной информации.
